🕒 Gradient Analog Clock 

A sleek, responsive analog clock built with HTML5, CSS3, and Vanilla JavaScript. 
This project features a vibrant purple gradient interface with high-contrast hands for easy readability and 
a modern dark-mode aesthetic.

🚀 Live Demo

Screenshot:(screenshot.png)

✨ Features

*Real-time Accuracy: Synchronized with your system clock using JavaScript’s Date object.

*Modern UI: Features a smooth radial gradient background and glowing clock hands.

*Minimalist Design: Clean typography and a centered layout for maximum focus.

*Fully Responsive: Scales beautifully across different screen sizes.

Built With

HTML5: Semantic structure for the clock face and developer info.

CSS3: Custom styling including Flexbox, CSS transitions, and complex gradients.

JavaScript (ES6): Logic to calculate angles for the hour, minute, and second hands.

 Logic Behind the Movement
 
 To keep the clock moving accurately, the following rotation logic was applied:
 Seconds: $360 / 60 = 6^\circ$ per second.
 Minutes: $360 / 60 = 6^\circ$ per minute.
 Hours: $360 / 12 = 30^\circ$ per hour
